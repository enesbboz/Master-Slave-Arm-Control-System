# Master-Slave-Arm-Control-System
Gerçek Zamanlı Kapalı Çevrim Master-Slave Robotik Replikasyon Sistemi
Bu proje; manuel olarak yönlendirilen bir referans kolunun (Master) hareketlerini, mutlak konum sensörleri verilerini kullanarak gerçek zamanlı ve yüksek hassasiyetle kopyalayan (Slave) kapalı çevrim bir robotik kontrol sisteminin donanım ve yazılım tasarımını içermektedir. Sistem, milisaniye mertebesinde gecikme (latency) ile çalışarak "teleoperasyon" ve "haptik geribildirim" teknolojilerinin temelini oluşturur.
